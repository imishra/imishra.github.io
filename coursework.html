Course Projects
  • Room Navigation using Deep RL
    This project propsed an approach of dummy reward where a re-inforcement learning agent receives a reward heuristic at each step or at a very short interval which guides the initial exploration so that the agent start reaching the goal early. This heuristic is designed in a way that it is as informative as possible but should go down as the agent becomes more and more confident about the policy in order to help facilitate enough exploration but should let the model learn the final policy without any perturbation to the reward system.

  • Text emotion classification on Spark
    A project focused on building a text emotion classifier which used the go emotion dataset with 28 different emotions.The spark environment was created using docker containers where each spark node is running as a separate container. The data was distributed to different machine which was then used to create a spark dataframe. Distributed random forest was used as the underlying algorithm to classify the emotions.
